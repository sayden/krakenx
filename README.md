Kraken-X
======================

[![Build Status](https://travis-ci.org/sayden/krakenx.svg?branch=develop)](https://travis-ci.org/sayden/krakenx)

A krakenjs based app with JSX templating and React with NoSQL persistence

### Setup
```sh
  npm install
```

### Build
```sh
  grunt build
```

### Test
```sh
	grunt test
```

### Run
```sh
  npm start
```
